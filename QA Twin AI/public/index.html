📖 How to get your Slack Bot Token:

  1. 1. Go to api.slack.com/apps → Create New App → From scratch
  2. 2. Enter app name → Select your workspace
  3. 3. "OAuth & Permissions" → Add Bot Token Scopes: channels:read, channels:history, users:read
  4. 4. "Install to Workspace" → Allow
  5. 5. Copy "Bot User OAuth Token" (starts with xoxb-...)
